1¬Ñ What is Git?

Ans:- Git is a distributed version control system designed to handle everything from small to very large projects with speed and efficiency. Here are the key aspects of Git:

 1. *Version Control*: Git tracks changes to files, allowing multiple people to work on the same project simultaneously without overwriting each other's work.

 2. *Distributed System*: Every user has a complete copy of the entire repository, including the full history of all changes.

 3. *Branching and Merging*: Git allows for the creation of branches, enabling parallel development. Branches can be merged back into the main branch, combining different sets of changes.

 4. *Commit History*: Git maintains a history of commits, each representing a snapshot of the project's state at a given time, complete with a commit message describing the change.

 5. *Collaboration*: Git is often used in conjunction with platforms like GitHub, GitLab, and Bitbucket, which provide additional tools for collaboration, such as pull requests, code reviews, and issue tracking.

        Overall, Git is a powerful tool for managing source code and collaboration in software development projects.
